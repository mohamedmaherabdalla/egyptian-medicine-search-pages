# Egyptian Medicine Search Demo

Static GitHub Pages version of the medicine search demo.

This public repository contains only the final static website and compact search catalog needed for the demo. The full research repository, reports, old datasets, experiments, and test-generation work remain private.

Live site:

```text
https://mohamedmaherabdalla.github.io/egyptian-medicine-search-pages/
```

This is a candidate-retrieval prototype, not a clinical decision system.
